## Chunk: platform_strategy_implications

**Tags:** platform_strategy, prioritization, viral_tech_2026

**Text:**
Because TikTok, Instagram Reels, and YouTube Shorts carry the majority of viral tech clips in the sample, strategy should prioritize publishing there first. Successful creative can then be ported to X and Facebook as proof, recaps, or carousels rather than primary bets. When generating hooks, intros, and CTAs, the system should assume TikTok-first framing by default, then adapt to Reels and Shorts with small timing and caption tweaks.

## Chunk: format_mix_overview

**Tags:** format_mix, content_formats, viral_tech_2026

**Text:**
The sample of viral tech videos is spread across multiple formats with no single dominant style. Formats include: AI tool demos, educational five-tip breakdowns, POV client stories, problem–solution reveals, before and after funnel teardowns, face-cam plus b-roll code walkthroughs, and faceless screen-share tutorials. Each format accounts for roughly a similar share of the sample, which means the system can rotate formats without sacrificing potential.

## Chunk: format_as_strategy_lever

**Tags:** format_strategy, hooks, video_system, viral_tech_2026

**Text:**
Format choice should be treated as a strategic lever. For awareness plays, the system should lean on fast AI demos and bold problem–solution reveals. For authority, POV client stories and before/after funnel teardowns work best. For hands-on education, face-cam code walkthroughs and faceless screen-share tutorials shine. When Claude or a local model generates a script or hook, it should select a format that matches the user’s stated goal and then apply hook patterns from the broader BADGR viral tech video system.

## Chunk: cross_platform_mapping

**Tags:** cross_platform, mapping, viral_tech_2026

**Text:**
Viral tech ideas should be reusable across TikTok, Instagram Reels, and YouTube Shorts with minimal changes: timing, caption length, and on-screen text density. The same underlying format—like a before/after funnel teardown—can be cut into a snappier hook for TikTok, a slightly slower walkthrough for Reels, and a more tutorial-focused version for Shorts. The RAG system should surface this cross-platform reuse idea whenever a user asks for multi-platform output for the same concept.
