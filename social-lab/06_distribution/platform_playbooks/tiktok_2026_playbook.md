# tiktok_2026_playbook

## Algorithm Rules (2026)

- **Saves and shares** carry 3–5x more weight than likes for distribution
- **Watch-through rate** (% of video watched) determines whether TikTok pushes the video beyond initial followers
- **Comments** boost the post, especially if the creator responds within the first 30 minutes
- **Hashtags:** 3–5 targeted hashtags outperform tag-stuffing; mix niche (#atlantalawfirm, #websitefix) with broad (#webdev, #techbusiness)
- **Trending audio:** using a trending sound in the first 5 seconds adds 2–5x reach boost — swap audio at posting time, not scripting time
- **Consistency:** TikTok rewards accounts that post on a predictable schedule; 3–5x/week minimum for growth

## Content Specs

| Spec | Value |
|------|-------|
| Aspect ratio | 9:16 vertical |
| Resolution | 1080 x 1920px minimum |
| Length | 15–30s for BADGR content (30–60s max) |
| Caption | 150 characters max for mobile view; CTA in first line |
| Hashtags | 3–5 per post |
| Cover frame | Set manually — show the hook text, not a blurry mid-frame |

- Peak windows: 4–6 PM and 8–10 PM EST (test both)
