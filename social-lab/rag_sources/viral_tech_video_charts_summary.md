# Viral Tech Video Charts Summary (BADGR 2026)

## Chart 1 – Distribution of Viral Tech Videos by Platform (Sample Set)

This chart shows how many viral short-form tech videos (in a small sample) came from each platform.

- TikTok: 7 videos
- Instagram Reels: 6 videos
- YouTube Shorts: 6 videos
- X (formerly Twitter): 3 videos
- Facebook: 2 videos

Key implications:
- TikTok currently leads the sample set for viral short-form tech videos and should be treated as the primary surface for new viral tech experiments.
- Instagram Reels and YouTube Shorts are close behind and should mirror TikTok patterns with light platform-specific edits (timing, caption length, on-screen text density).
- X and Facebook produce fewer viral tech videos in this sample and should be treated mainly as secondary surfaces for amplification, proof, and recaps rather than as primary test beds for new concepts.

## Chart 2 – Content Format Mix in Viral Tech Videos (Sample Set)

This pie chart shows the mix of content formats used across the viral tech videos in the sample. Each format accounts for roughly 16–20% of the total, meaning no single format dominates.

Formats included:
- AI tool demo (faceless)
- Educational 5‑tip breakdown
- POV story: client result
- Problem–solution reveal
- Before/after funnel teardown
- Face cam + b‑roll code walkthrough
- Faceless screen share tutorial

Key implications:
- Viral tech content succeeds across multiple formats; you are not locked into a single style.
- Rotating formats (demos, POV stories, teardowns, walkthroughs, faceless tutorials) keeps the feed fresh while staying on brand.
- Format choice should be treated as a strategic lever:
  - Awareness: fast AI demos, bold problem–solution reveals.
  - Authority: POV client stories, before/after funnel teardowns.
  - Hands-on education: face-cam code walkthroughs, faceless screen-share tutorials.
- For any given idea, the system can pick one of these formats based on platform and goal (awareness, authority, lead generation) and then adapt it across TikTok, Reels, and Shorts with minimal changes.
